ls(list) - выводит список содержимого в папке включая папки
cd (change directory) - переход между папками
cd ../ -переход в одну папку назад
control+ l очистить терминал
clear очистить терминал
pwd(path working directory)- абсолютный путь к папке в которой вы находитесь
ls-a  показать содержимое папки включая скрытые папки
rm -rf .git вырубает все
git init инициаливация git репозитории
git remote add origin <ssh url>- привязка локального репозитория к удаленномк
git remote -v - посмотреть какому репозитории привязан локально
git remote remove <name>- удалить привязку
git add . - упоковать
git commit -m 'Some messege' - маркировка
$ git push origin master отправка данных
